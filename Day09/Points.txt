REST API

Methods

GET method
----------
GET is one of the most common HTTP methods used when working with API, it is used to retrieve resources from a given API. GET is readonly

POST Method
-----------
POST is used to add data to a REST API to create a new resource.

PUT Method
----------
PUT method is used to update the existing todo data new data

PATCH Method
------------

DELETE Method
-------------
